the definition of Operating Systems

The operating system, and its abbreviation (OS): is a set of programs responsible for managing resources (computer hardware), computer software, and represents an intermediary between the user and the computer. In another definition, the operating system represents a bridge to run user programs, and it performs basic tasks such as: management and allocation Computer resources (memory, hard disk, access to peripherals etc ...), ordering priority to handle orders, controlling input and output devices such as: the keyboard, as well as to facilitate networking, and file management.
