the Types of operating system

There are many types of operating systems. This is due to the fact that computer uses and applications have become required in various areas of life in a very large way, and this great demand for computers and its applications has opened the way for many operating systems that meet specific needs according to the nature and quality of their programming; Each operating system has its own advantages and characteristics only. Among the operating systems used in the computer world are:

1-(DOS Operating System): The abbreviation for Disk Operating System. This system is one of the very old systems used to run many device programs. For example, this system was used to operate IBM personal devices, in addition to IBM devices, and this system was called the company name, In 1981, the president of Microsoft Corporation, Bill Gates, developed this system, called the developed system MS-DOS, and this system relies on running direct text commands. This system is single-tasked, and it should be noted that it is the core of Microsoft Corporation.

2-(LINUX operating system): It is one of the most famous, widespread and used systems, and it is an open source system, and it is characterized by a beautiful and simple graphic interface, and it has many different versions, and is characterized by high protection and safety when using it, and it can be modified because it is a completely free system , And is usually used by protection engineers in particular and networks in general.

3-(Windows operating system): This system is one of the most famous and prominent operating systems, and it belongs to Microsoft International, and the beginning of this system was in 1985 AD. Where it was a graphical board to help write commands in the DOS system, and in the year one thousand nine hundred and ninety-five, Microsoft launched an independent operating system that it called Windows 95, and in the year one thousand nine hundred and ninety-eight the company launched a new version of the operating system, called Windows 98 Many different versions of Windows were released after that, such as Windows 2000, Windows 2003, Windows 7, Windows Vista, Windows 8, and finally Windows 10.
